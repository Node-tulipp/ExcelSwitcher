# Excel Switcher

<img width="486" height="593" alt="image" src="https://github.com/user-attachments/assets/42c0edf0-9be7-47ad-bdda-e7479d80e7fc" />


複数のExcelファイルを素早く切り替えるための軽量ツール

## 特徴

- 🚀 タブ化せずに複数のExcelを管理
- 📁 グループ機能で整理
- 💾 設定の自動保存
- 🪶 常駐不要の軽量設計

## 使い方

1. `ExcelSwitcher.exe` を起動
2. Excelファイルをダブルクリックで切り替え
3. ドラッグ&ドロップでグループ管理

### 操作方法

- **ダブルクリック**: Excelウィンドウをアクティブ化
- **Enter**: 選択中のExcelをアクティブ化
- **Esc**: ツールを終了
- **右クリック（空白）**: 新しいグループを作成
- **右クリック（グループ）**: グループを削除/名前を変更
- **ドラッグ&ドロップ**: ファイルやグループの並び替え

## 動作環境

- Windows 10/11
- .NET 9.0 Runtime（通常はインストール不要）

## 開発環境

- LibreAutomate
- C# / .NET 9.0
- Windows Forms
