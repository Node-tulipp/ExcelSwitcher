# Excel Switcher

<img width="486" height="593" alt="image" src="https://github.com/user-attachments/assets/42c0edf0-9be7-47ad-bdda-e7479d80e7fc" />


複数のExcelファイルを素早く切り替えるための軽量ツール

## ✨ 特徴

- 🚀 タブ化せずに複数のExcelを管理
- 📁 グループ機能で整理
- 💾 設定の自動保存
- 🪶 常駐不要の軽量設計
- 🔄 リアルタイム更新
- ⚡ 連続操作対応（フォームを閉じずに複数のExcelを切り替え可能）

## 📥 ダウンロード

[最新版をダウンロード（Releases）](../../releases/latest)

## 🎯 使い方

### インストール

1. [Releases](../../releases/latest)から`ExcelSwitcher-v1.0.1.zip`をダウンロード
2. ZIPファイルを解凍
3. フォルダ内の`ExcelSwitcher.exe`をダブルクリックで起動

### 基本操作

- **ダブルクリック**: Excelウィンドウをアクティブ化（フォームは開いたまま）
- **Enter**: 選択中のExcelをアクティブ化
- **Esc**: ツールを終了

### グループ管理

#### グループの作成
- **未分類グループを右クリック** → 「新しいグループを作成」

#### グループの管理
- **グループ名を右クリック** → 「グループを削除」または「グループ名を変更」
- **ドラッグ&ドロップ** → グループの順序を変更

#### ファイルの移動
- **ファイルをドラッグ&ドロップ** → グループ間で移動

## ⚙️ 動作環境

- Windows 10/11
- .NET 9.0 Runtime（通常はインストール不要）

## ⚠️ 既知の制限

- TreeViewコントロールの空白エリア（ノードがない下部）では右クリックメニューが表示されません
  - これはWindows FormsのTreeViewコントロールの仕様です
  - **回避策**: 「未分類」グループを右クリックして新しいグループを作成できます
